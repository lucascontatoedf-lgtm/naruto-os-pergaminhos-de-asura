# Documentation

Artefatos de portfólio do projeto **Naruto: Os Pergaminhos de Asura** (MVP Vertical Slice em Godot 4).

Vídeos de gameplay, screenshots e GIFs que demonstram o estado do projeto em cada milestone:

- **Semana 1 — Core Gameplay**: movimentação, pulo (coyote + buffer + double jump), FSM, chakra, HUD de debug, kill zone.
- **Semana 2 — Level Design**: Vila da Folha, Floresta da Névoa, NPCs, segredos.
- **Semana 3 — Boss + Polish**: Zabuza, áudio, partículas, otimização final.

## Organização sugerida

```
documentation/
├── videos/        # gravações .mp4 de gameplay
├── screenshots/   # capturas .png / .jpg
└── gifs/          # clips animados .gif
```

> Mantenha cada arquivo abaixo de ~10 MB para navegação rápida na interface do GitHub. Para vídeos maiores, hospede no YouTube/Vimeo e linke aqui.
